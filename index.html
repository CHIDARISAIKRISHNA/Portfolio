Projects


Student Outing Management System

It is a platform designed to manage student outing details, including personal information and registration numbers. It allows watchmen to log student exits and send 9:00 PM reminders. Students can access their information and faculty contact details, ensuring easy communication. Additionally, the system connects current students with alumni, facilitating discussions about placements and internships. This platform enhances campus safety, streamlines communication, and provides valuable resources to students and faculty.

Insurance Management System

It is a web-based application designed to streamline and manage various operations within an insurance company. It offers role-based access for customers, HR personnel, finance staff, inspectors, and agents, providing tailored functionalities for each user. The system utilizes MySQL for efficient database management, storing critical data such as insurance policies, customer information, and claims. Built with Node.js, Express.js, and EJS, the system dynamically generates HTML pages for a seamless user experience while ensuring secure handling of sensitive information.

Dynamic Restaurant Ordering and Payment System

Developed an interactive menu platform that enhances customer experience by allowing easy browsing and item selection across categories like Breakfast, Lunch, and Dinner. The system features real-time cost updates with GST, ensuring accurate pricing, and includes a digital payment receipt that provides an itemized breakdown of costs and taxes. This project focuses on a seamless, user-friendly interface to improve the ordering and payment experience.
